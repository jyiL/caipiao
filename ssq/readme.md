### 说明：

- 本程序是借助node http模块获取网页内容，然后使用cheerio进行文本处理，原理简单。
- http请求是属于异步请求，而本程序为了避免异步带来的数据混乱问题，改成同步进行，所以运行速度会很慢，请耐心等待。（之前使用过异步获取，速度非常快，只是需要在后期重新整理数据）
- 本程序纯属个人兴趣练手程序，程序存在许多不足的地方，欢迎指出。
- 拿到的数据可以自行作为参考数据测试训练等，严禁用作商业用途！
- 文件写入的数据格式是： （按最新一期排序）

    data = {
    	balls: [
    		{
    			issueNumber: '',      // 期号
    			redBall: [],          // 红球
    			blueBall: ''          // 蓝球
    		}
    	]
    }
